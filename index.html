How It Works
01
Pyodide Loads Python in Your Browser
When the page opens, it loads Pyodide — a full CPython interpreter compiled to WebAssembly. This lets real Python code run inside your browser with zero server or installation needed.
02
AHP Derives Criterion Weights
The Analytic Hierarchy Process (Saaty, 1970s) takes a 5×5 expert pairwise comparison matrix, normalizes it column-wise, and computes row means as priority weights. Consistency is validated with CR = CI / RI < 0.10.
03
TOPSIS Ranks the Alternatives
TOPSIS (Hwang & Yoon, 1981) normalizes the decision matrix, applies AHP weights, identifies positive/negative ideal solutions, computes Euclidean distances D+ and D−, and derives a closeness coefficient C = D− / (D+ + D−).
04
Sensitivity Analysis Tests Robustness
Each criterion weight is perturbed by +20% (others rescaled proportionally) and TOPSIS is re-run. If the top-ranked alternative stays the same across all perturbations, the decision is robust.
05
Output is Rendered in the Terminal
ANSI escape codes in the Python output are parsed by JavaScript and converted to colored HTML spans. The result is a colour-coded terminal replica rendered directly in the browser.
